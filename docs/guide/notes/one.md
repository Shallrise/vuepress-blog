> #### 新建vue项目

1. vue create mock-demo

> #### 安装依赖

```
使用 axios发送ajax

  cnpm install axios --save

使用mockjs产生随机数据

  cnpm install mockjs --save-dev

使用json5解决json文件按，无法添加注释问题

  cnpm install json5 --save-dev
```

<u>若npm安装出现</u>

```
verb npm-session
```

<u>使用淘宝镜像解决</u>

```
npm config set registry http://registry.npm.taobao.rog
```
